分布式组件-SpringCloud Alibaba-Nacos注册中心

官网地址：https://github.com/alibaba/spring-cloud-alibaba/blob/master/spring-cloud-alibaba-examples/nacos-example/nacos-discovery-example/readme-zh.md

启动Nacos Server

  1. 首先需要获取Nacos Server ,直接现在安装即可，官网地址：https://github.com/alibaba/nacos/releases

  2. 启动 Server，进入解压后文件夹或编译打包好的文件夹，找到如下相对文件夹 nacos/bin，并对照操作系统实际情况之下如下命令。

    使用说明

  3. 修改pom.xml文件，引入Nacos Discovery Starter

  1. 在应用的 /src/main/resources/application.properties 配置文件中配置 Nacos Server 地址
  1. 使用@EnableDiscoveryClient 注册开启服务注册于发现服务
  1. 增加配置，在服务的 /src/main/resources/application.yml中添加基本配置信息
  1. nacos访问地址http://10.2.1.120:8848/nacos,服务管理-服务列表中存在上述配置服务则发布成功
