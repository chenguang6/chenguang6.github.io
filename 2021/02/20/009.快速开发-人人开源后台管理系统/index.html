快速开发-人人开源搭建后台管理系统

人人开源介绍

  • 我们使用人人开源项目，进行商城后台的快速开发
  • 开源项目路径：https://gitee.com/renrenio
  • renren-fast 后台项目，renren-fast是一个轻量级的Spring Boot2.1快速开发平台，其设计目标是开发迅速、学习简单、轻量级、易扩展；使用Spring Boot、Shiro、MyBatis、Redis、Bootstrap、Vue2.x等框架，包含：管理员列表、角色管理、菜单管理、定时任务、参数管理、代码生成器、日志管理、云存储、API模块(APP接口开发利器)、前后端分离等
  • renren-fast-vue 前端项目，renren-fast-vue 是基于vue、element-ui构建开发，实现renren-fast后台管理前端功能，提供一套更优的前端解决方案。
  • renren-generator逆向生成项目，人人开源项目的代码生成器，可在线生成entity、xml、dao、service、vue、sql代码，减少70%以上的开发任务

导入renrenfast的数据库

  • 数据库sql在项目文件夹，db目录下，我们使用mysql数据库
  • 在我们数据库中创建 lightmall_admin ,作为我们后台服务的数据库，将人人项目的sql复制出来在mysql中执行

谷粒商城项目整合 renrenfast

  • 将我们renren代码拉到我们聚合工程目录下，lightmall-project

  • 聚合工程 pom.xml 中添加 renren-fast

  • 修改renren-fast的数据源，为我们服务器地址，数据库账号密码为我们创建数据库是设置的账号密码

  • 此时renren-fast就可以正常启动了

renrenio前端地址

依赖环境

  • node js , 主要用的npm（包管理工具），类似于maven，用于下载相关依赖，自行进行下载
  • 验证本机nodejs,cmd命令，执行node -v ,提示版本号则安装成功

运行前端项目

  • 使用VSCode打开我们前端项目

  • 执行 npm install ，进行依赖包的下载，下载的依赖会在node_modules目录下

  • 启动前端

  • 成功启动，即可使用，账号密码均为admin

谷粒商城项目整合 renrenio逆向工程

首先下载renren逆向工程代码

  • renren-generator，可直接在gitee上搜索renren

将人人开源逆向工程项目，拉入到我们项目文件夹下

将 renren-generator 依赖到聚合工程

逆向工程使用

  • 修改数据库连接地址，application.yml，将连接地址，账号密码修改为实际需逆向代码的工程数据库地址

  • 修改配置信息，generator.properties，根据实际情况进行配置

  • 启动逆向工程项目，默认端口为80，启动成功后访问项目地址:localhost:80

  • 我们可勾选所有字段，点击生成代码，生成成功后，会自动下载一个压缩文件

  • 压缩文件中，为生成的所有代码，包括controller、service、dao、entity、mapper、vue页面等

  • 将生成的代码，放入我们项目指定工程下，之后会发现有异常，因为我们没有对应的工具类，我们可以从renren-fast中找到并引用

  • 因为这些工具类，所有的工程都需要使用，所以我们创建一个common工程(maven)，存放公共的工具类及依赖

  • 注:

    逆向工程生成的Controller，方法上会存在@RequiresPermissions（Shiro权限框架），如果不需要可以修改逆向工程模板，resources - > template -> Controller.java.vm,将不需要的内容进行删除或注释即可

    其他几个服务也已同样的方式进行逆向生成
