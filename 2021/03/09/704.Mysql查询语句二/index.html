Mysql查询语句-第二讲

常见函数：

字符函数：

​ length ：获取字符串字节数函数
​ concat：合并字符串函数
​ substr：字符串截取
​ instr：字符查找函数
​ trim：去除左空格函数
​ upper ：字母大写转换函数
​ lower ：字母大写转换函数**
​ lpad：填充字符串左边
​ rpad：填充字符串右边
​ replace：字符串替换

字数函数：

​ round：四舍五入函数
​ ceil：向上取整函数
​ floor : 向下取整函数
​ truncate : 数值截取函数
​ mod : 取模函数

日期函数

​ now : 获取当前日期时间
​ crudate ：获取当前日期
​ curtime：获取当前时间
​ year：从日期中选择出周数
​ month：从日期中选择出月份数
​ monthname：指定日期的月份的全名
​ day：返回给定日期的月份的
​ hour：从时间中选择出小时数
​ second：返回时间秒
​ str_to_date：将时间格式的字符串
​ date_format：将日期值格式化为特定格式

其他函数

​ version：查看mysql版本号
​ database
​ user

排序查询

  • 语句：select 查询列表 from 表 where 赛选条件 order by 排序列表 【asc|desc】
  • 特点：
        1、asc代表的是升序，desc代表的是降序；如果不写默认为升序
      2、order by 子句中可以支持单个字段、多个字段、表达式、函数、别名
      3、order by 字句一般是放在查询语句的最后，但limit子句除外

常见函数

  • 概念：类似于java中的方法，将一组逻辑语句封装在方法体中，对外暴露方法名
  • 好处： 1、隐藏了实现细节 2、提高代码的重用性
  • 调用：select 函数名（） 【from 表名】；
  • 特点：
    1. 叫什么（函数名）
    2. 干什么（函数功能）
  • 分类：
    1. 单行函数，如：concat、length，ifnull等
    2. 分组函数，功能：做统计使用，又称为统计函数、聚合函数、组函数
