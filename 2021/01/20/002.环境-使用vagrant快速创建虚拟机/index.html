环境使用vagrant快速创建Linux虚拟机

  1. 介绍
  • Vagrant 是一个基于Ruby的工具，主要用于创建和部署虚拟化开发环境。它以来于Oracle的开源VirtualBox虚拟化系统，通过使用 Chef创建自动化虚拟环境。
    • vagrant 可以快速，方便，全自动的构建虚拟化环境，这也是我们选择它的原因，而不是让我们像以前一样全部自己来部署。
    • 它类似与 docker 这种，有自己的仓库，我们直接可以通过命令从仓库中拉取虚拟镜像来快速构建

  1. Vargrnat基本命令

    命令 作用
    vagrant init 初始化box的操作，会生成vagrant的配置文件Vagrantfile
    vagrant up 启动本地环境
    vagrant ssh 通过 ssh 登录本地环境所在虚拟机
    vagrnat relod 重启虚拟机

  1. 安装Linux虚拟机

    • 这里使用virtualbox，官网下载地址 https://www.virtualbox.org/wiki/Downloads

    • 选择windows版本的进行下载

    • 傻瓜式一键安装

  2. 安装Vagrant

    • 简介

      结合VirtualBox快速创建虚拟机

    • Vagrnat官网地址：https://www.vagrantup.com/，官网页面，点击 Download ，进行下载Windows版本

    • 傻瓜式下一步安装，安装完成后重启我们的电脑

  3. 验证vagrant是否安装成功

  4. cmd中输入vagrant如果有提示则安装成功

    ​

  5. 使用vagrant创建虚拟机

    • vagrant环境官网：https://app.vagrantup.com/boxes/search

    • cmd命令窗口，vagrant init centos/7,进行初始化环境；（centos/7 为vagrant环境官网中的名称）

    • vagrant up,进行下载镜像并启动虚拟容器

    • vagrant ssh 进行连接虚拟机

    • exit 退出

  6. 网络IP配置

    • 默认虚拟机IP不是固定IP，开发不方便，所以设置一个固定的IP

    • 修改 vagrantfile,默认在用户文件夹中

      1. 首先我们查看自己 Windows 电脑的 IP，其中有个网卡 VirtualBox Host-Only Network。，这就是虚拟机的网卡，看到其IP地址段

      2. 直接在我们刚才 vagrant 的目录下的 Vagrantfile 文件中就行配置修改，这是我们刚才创建的虚拟机的配置文件 ，配置 config.vm.network "private_network", ip: "192.168.56.10"

  7. 异常

    • 报错详情

      原因：可能因为网络的原因，下载virtualbox.box失败，可以手动进行下载

  8. 手动下载BOX

    • 上官网上下载需要安装的系统，https://app.vagrantup.com/boxes/search

    • 将下载好的box文件存放到你要安装的目录下，在该目录下使用cmd命令

    • 添加 vagrant box 到 box list

    • 初始化一个虚拟机使用刚才添加的 vagrant box

    • 启动 vagrant box 虚拟机

    • 执行ssh进行登录
