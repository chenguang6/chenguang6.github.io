501-自增变量说明

January 20, 2021 863

自增变量

  1. 题目

  2. 结果

    • i = 4,j = 1,k = 11

  3. 分析该题

    • i = i++;

      1. 把 i 的值压入操作数栈
      2. i 变量自增1
      3. 把操作数栈中的值赋值给i
      4. 故 i = 1

    • int j = i++;

      1. 此时 i =1;

      2. 把i的值压入操作数栈

      3. i变量自增1

      4. 把操作栈中的值赋值给j

      故j = i

    • int k = i + ++i * i++;

      1. 此时 i = 2;
      2. 把 i 的值压入操作数栈，故数据栈中i=2
      3. （++i）i 变量自增1
      4. （++i）把 i 的值压入操作数栈,故数据栈中i=3
      5. ( i++) 把 i 的值压入操作数栈，此时 i=3
      6. ( i++) i 变量自增1,故操作栈中i=3，局部变量i=4;
      7. (++i * i++) 把操作数栈中前两个弹出求乘积结果再压入栈,故此时操作栈中 i 均为3，乘积为9
      8. （i + ++i * i++）把操作数栈中的值弹出求和再赋值给k，i从开始赋值为操作栈中数据为2，(++i * i++) 乘积为9，故2 + 9 = 11；

  4. 总结

    • 赋值=，最后计算
    • =右边的从左到右加载值一次压入操作栈
    • 实际先算哪个，看运算符优先级
    • 自增、自减操作都是直接修改变量的值，不经过操作栈
    • 最后的赋值之前，临时结果也是存储在操作数栈中
