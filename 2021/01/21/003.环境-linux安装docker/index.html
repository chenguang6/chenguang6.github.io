环境-虚拟机网络设置

  1. ​ Docker 简介

    • 虚拟化容器技术。Docker基于镜像，可以秒级启动各种容器。每一种容器都是一个完成的运行环境，容器之间相互隔离
    • docker 镜像网站：https://hub.docker.com/
    • docker 官网文档：https://docs.docker.com/
    • docker centos 安装文档地址：https://docs.docker.com/engine/install/centos/

  2. docker 常用命令

    命令 作用
    docker -v 查看Docker版本
    docker image 检查当前虚拟机下载镜像
    docker ps 查看运行程序
    docker exec -it * /bin/bash 进入*容器内部
    docker restart * 重启
