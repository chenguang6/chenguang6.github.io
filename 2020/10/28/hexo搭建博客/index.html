hexo搭建个人博客

  1. 依赖环境

    • node，查看node版本

    • npm ，node自带，查看npm版本

    • cnpm 淘宝镜像（国内镜像，加快install速度）

    • hexo 框架

  2. 创建博客,启动博客

    • 首先，创建博客目录,cmd到此目录下

    • 初始化博客

    • 启动博客

    • 启动成功后，控制台会存在本地访问路径，为http://localhost:4000/

  3. 添加博客文档

    • 创建博客文档

    • 博客文档创建路径：source/_posts/

    • 编辑博客文档,vim方式进行编辑，编辑之后进行wq（保存）

    • 重新生成，回退到博客跟目录

    • 清理

    • 重新生成

    • 重新启动

      至此，搭建，本地发布完成

  4. 博客部署到github上

    • 新建一个仓库，名称命名为： github用户名.github.io，必须以这种格式进行命名

    • 博客根目录创建 github插件

    • 配置 _config.yml，最下方三个属性进行修改

    • 部署至远端（github）

      提示输入github的账号密码，自动发布到github上，github仓库中会存在发布的内容
      访问仓库名称：chenguang6.github.io，就可以访问到自己创建的博客啦！！

  5. 自定义更换主题

    • 可上 https://hexo.io/themes/ 官网上进行提先心仪的主题，点击图片进行预览，点击名称跳转至github地址上
    • 下载挑选完成的主题,指定目录themes/liveforcode（liveforcode可自行更换）
    • 配置到自己的博客中，修改 _config.yml 文件,theme:更换为上述的名称名liveforcode
    • 重新清理
    • 重新生成
    • 重新启动
